ocean_doc
====
用于存放深海标准产品文档



# 文档命名

产品文档名称：深海-产品序号-产品中文名称-文档类型-版本号

> 例1：深海-05-数据集成平台-产品白皮书-V1.0.docx
>
> 例2：深海-09-数据可视化平台-用户操作手册-V1.0.docx
>
> 例3：深海-09-标签画像平台-概要设计说明书-V1.0.docx

> 例4：ocean-10-tag-manager_sge_v1.0.sql
